Simple Calculator

A basic **calculator web application** built using **HTML, CSS, and JavaScript**.  
It performs standard arithmetic operations like addition, subtraction, multiplication, and division.

This project is beginner-friendly and great for practicing DOM manipulation and JavaScript logic.

Features
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)
- Clear (C) button
- Responsive and simple UI
- 
Technologies Used

- HTML – structure  
- CSS – styling  
- JavaScript – functionality
